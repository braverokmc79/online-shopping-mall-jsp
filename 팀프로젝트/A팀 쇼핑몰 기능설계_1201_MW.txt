A팀 "쇼핑몰 프로젝트" 기능 설계

1. 회원관리
     - 비밀번호 찾기 : 난수 생성하여 이메일로 발송 -> 사용자가 확인 후 입력하여 인증 -> 비밀번호 변경
     - 회원별 포인트 관리 : 가입시 2,000 포인트 / 구매 시 상품금액의 5% 포인트 부여 -> 싸이트 내 현금으로 사용가능
##                               리뷰 작성 시 100 포인트 부여 / 사진 리뷰 작성 시 300 포인트 부여
   
2. QnA
     - 게시판 형 (일반)
     - 1:1 채팅 형

## 3. 구매
##     - 장바구니 활용해 다양한 제품 한 번에 구매 가능
##     - 결제 정보 및 수취인 주소 저장 기능
##     - 구매한 제품과 유사한 제품 추천 기능
##     - 비회원 구매 기능 *


## 4. 리뷰
##     - 주문, 품목 당 1개의 리뷰 작성 기능
##     - 사진 게시 
##     - 별점 표시 (조회 시 각 별점 별로 글 조회 가능)

5. 관리자모드
     - 제품 관리
          - 제품 master list 관리 (테이블 형식)
          - 입고 그룹별 재고 통합 관리
##	   (식품 특성 상 입고일, 유통기한, 폐기 재고 관리 필요  -->  재고 소진율 조회 기능)
##          - 판매 제품 관리
##               (재고 중에서 판매원하는 상품 선택 및 게시 사진 및 글 등록 기능)
